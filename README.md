# Assam Don Bosco University Bus Tracker

## Project Overview

The **University Bus Tracker** is a web-based application developed using PHP and MySQL.  
It helps students easily check university bus routes, timings, and availability, reducing waiting time and improving transportation efficiency.

This project was developed as part of a group assignment following professional development practices.

## Features

### User Features
- View available buses  
- Check bus routes and stops  
- View bus timings  
- Track bus availability  

### Admin Features
- Add, update, and delete buses  
- Manage routes and timings  
- Monitor system data  

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server Environment:** XAMPP

## Requirements

- XAMPP (Apache & MySQL)
- Web Browser (Google Chrome recommended)

---

## Installation & Setup

### 1. Copy the project folder to:
   ```
   C:/xampp/htdocs/
   ```
   
### 2. Start XAMPP
   Open XAMPP Control Panel
   Start Apache and MySQL
   
### 3. Setup Database
   #### Open browser and go to:
     http://localhost/phpmyadmin

   #### Create a database named:
     bus_tracker
     
   #### Click import
   Import the SQL file from the database folder

### 4. Run the Project
   #### Open browser and go to:
     http://localhost/bustracker
